# RekenRace

RekenRace is een interactieve rekengame voor kinderen. De speler lost rekensommen op om een raceauto naar de finish te laten rijden.

## Functies

- Keuze uit groep 5, 6, 7 en 8
- Keuze uit moeilijkheid: makkelijk, normaal en moeilijk
- Rekensommen met optellen, aftrekken, vermenigvuldigen en mix
- Score, timer en voortgang
- Levelsysteem
- Adaptieve moeilijkheid
- Geluidseffecten en achtergrondmuziek
- Visuele racebaan met auto en finish
- Instructieknop voor uitleg

## Gebruikte technieken

- HTML
- CSS
- JavaScript
